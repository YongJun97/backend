// 주제
온라인 쇼핑몰: 자바를 활용하여 
상품 목록, 장바구니, 결제 처리 등을 지원하는 온라인 쇼핑몰을 개발할 수 있습니다.

// 목표
상품 목록을 1,2,3, 과 같이 정수로 입력받고 선택한 상품을 결제를 할지 와 장바구니에 넣을지
선택지를 나눠놓고 결제를 눌렀을 때는 결제가 완료 되었다는 문구를 띄우고 장바구니를 선택
했을때는 데이터를 목록에 저장 후 상품이 장바구니에 담겼다는 문구를 띄움.
장바구니에 담긴 목록을 삭제하는 프로그램을 추가하고 삭제가 된다면 상품을 장바구니에서 삭제
했다는 문구를 띄움

1. 상품 목록을 정수로 입력받아 선택
2.결제 , 장바구니 선택
3. 결제완료 문구, 장바구니에 담고 완료 문구
4.장바구니 목록 삭제와 삭제완료문구
5.종료 및 종료문구

// 필요한 생성자
상품의 이름
상품의 가격
상품의 별점
상품의 리뷰의 수
